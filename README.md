This repository houses the digital assessment for the At the Airport: Part 1 unit.

📝 Project Overview

This assessment is designed for ESL learners (NRS Levels 1-5) to test their comprehension of airport security procedures and technical vocabulary based on the Ellii "At the Airport" curriculum.

🚀 How to Use

Student Access: Students can access the assessment via the GitHub Pages link provided in class.

Device Compatibility: Optimized for mobile phones to allow for in-class "Bring Your Own Device" (BYOD) assessment.

Content:

Part 1: Comprehension (True/False) based on the Martin & Ellen dialogue.

Part 2: Vocabulary Review (Multiple Choice) focusing on technical travel terms (screening bin, unattended, boarding pass).

📊 Compliance & Reporting

Educational Standard: ELPS 10.4 & NRS Level Alignment.

Time Credit: 3.25 Hours (as part of the full April 20th lesson block).

Assessment Data: Used to verify "Measurable Skill Gains" for SEP folder documentation.
